### Tecnologias utilizadas

- Angular 17+
- TypeScript
- HTML
- TailwindCSS
- VS Code

### Pré-requisitos
Antes de rodar o projeto, verifique-se de ter instalado:
- Node.js
  - você pode verificar se já está instalado usando "node -v" no terminal
- Angular CLI
  - caso não tenha, instale com "npm install -g @angular/cli"

### Como rodar

1. Clone o repositório GitHub: 
   - git clone https://github.com/Luis-coelho30/limpai-frontend.git

2. Abrir o projeto no VS Code e entrar na pasta "frontend":
    - cd limpai-frontend/frontend

3. Rodar no terminal do próprio VS Code:
    - ng serve

4. Um link para o localhost será exibido no terminal do VS Code. Abri-lo te levará para a interface gráfica do projeto.
